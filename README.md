# tomato-clock

**番茄时钟.**

## To Use

```bash
# Clone this repository
git clone https://github.com/TracyYJZ/tomato-clock.git
# Go into the repository
cd tomato-clock
# Install dependencies
npm install
# Run the app
npm start
```
